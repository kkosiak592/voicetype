Create `src-tauri/src/foreground.rs` with:

1. **Types (top of file):**
   - `DetectedApp` struct: `exe_name: Option<String>`, `window_title: Option<String>`. Derive Clone, Serialize. This is the return type of detect_foreground_app.
   - `AppRule` struct: `all_caps: Option<bool>`. Derive Clone, Serialize, Deserialize, Default. The Option<bool> provides three-state: None=inherit, Some(true)=force ON, Some(false)=force OFF.
   - `AppRulesState` struct: `pub AppRulesState(pub std::sync::Mutex<HashMap<String, AppRule>>)`. This follows the ActiveProfile pattern.

2. **Detection chain (`pub fn detect_foreground_app() -> DetectedApp`):**
   - Call `GetForegroundWindow()`. If null HWND, return DetectedApp { exe_name: None, window_title: None }.
   - Call `GetWindowThreadProcessId(hwnd, &mut pid)`. If pid is 0, return empty DetectedApp.
   - Get window title via `get_window_title(hwnd)` helper.
   - Get exe name via `get_process_exe_name(pid)` helper.
   - If exe name is "applicationframehost.exe", attempt `resolve_uwp_child(hwnd)`. If UWP resolution returns Some, use that; otherwise keep "applicationframehost.exe".
   - Return DetectedApp with the resolved exe_name and window_title.

3. **Helper: `unsafe fn get_process_exe_name(pid: u32) -> Option<String>`:**
   - `OpenProcess(PROCESS_QUERY_LIMITED_INFORMATION, false, pid)` — use `.ok()?` to handle access denied gracefully.
   - Allocate `[0u16; 260]` buffer (MAX_PATH).
   - `QueryFullProcessImageNameW(handle, PROCESS_NAME_FORMAT(0), PWSTR(buf.as_mut_ptr()), &mut size)`.
   - **CRITICAL:** Call `CloseHandle(handle)` before any return (explicitly, do not rely on Drop).
   - Extract filename from full path with `std::path::Path::new(&full_path).file_name()`.
   - `.to_string_lossy().to_lowercase()` on the filename.

4. **Helper: `unsafe fn get_window_title(hwnd: HWND) -> Option<String>`:**
   - `[0u16; 512]` buffer.
   - `GetWindowTextW(hwnd, &mut buf)` returns length.
   - If len > 0, return `String::from_utf16_lossy(&buf[..len as usize])`.

5. **UWP resolution: `fn resolve_uwp_child(parent_hwnd: HWND) -> Option<String>`:**
   - Define `unsafe extern "system" fn enum_child_proc(hwnd: HWND, lparam: LPARAM) -> BOOL` callback.
   - Callback gets child PID, calls `get_process_exe_name`, and if the name is not "applicationframehost.exe", stores it via LPARAM and returns BOOL(0) to stop enumeration.
   - Call `EnumChildWindows(Some(parent_hwnd), Some(enum_child_proc), LPARAM(&mut result as *mut _ as isize))`.
   - Return the result Option<String>.

6. **Unit tests (`#[cfg(test)] mod tests`):**
   - `test_app_rule_serde`: Verify AppRule with all_caps: Some(true), Some(false), and None all round-trip through serde_json.
   - `test_app_rules_map_serde`: Verify HashMap<String, AppRule> round-trips, including empty map.
   - `test_detected_app_serialize`: Verify DetectedApp serializes correctly (exe_name and window_title as expected JSON).
