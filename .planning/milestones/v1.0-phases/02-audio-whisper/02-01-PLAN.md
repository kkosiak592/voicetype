**Create src-tauri/src/audio.rs** implementing:

1. **AudioCapture struct** holding:
   - `_stream: cpal::Stream` (must stay alive for persistent capture)
   - `recording: Arc<AtomicBool>` (flag toggled by hotkey)
   - `buffer: Arc<Mutex<Vec<f32>>>` (accumulated 16kHz mono samples)

2. **start_persistent_stream() -> Result<AudioCapture, ...>** function:
   - Get default host and default input device via `cpal::default_host()`
   - Read `device.default_input_config()` to get native sample rate and channel count
   - Log device name, native sample rate, and channel count at info level
   - Create rubato `FftFixedIn::<f32>::new(native_rate, 16000, chunk_size, 2, 1)` resampler
   - Use chunk_size = 1024 (approximately 64ms at 16kHz)
   - Wrap resampler in a struct with staging buffer for chunked accumulation (the callback delivers variable-length buffers but rubato needs fixed chunks)
   - `build_input_stream` callback:
     a. Check `recording.load(Ordering::Relaxed)` — return immediately if false
     b. Downmix to mono: average all channels per frame
     c. Push mono samples into the staging buffer
     d. When staging has >= chunk_size samples, drain chunks through resampler
     e. Extend the main buffer with resampled 16kHz output
     f. Use `try_lock()` on buffer mutex (never blocking send in audio callback — see Pitfall 2 in RESEARCH.md)
   - Call `stream.play()` immediately — stream is always running
   - Log "Audio stream started: {native_rate} Hz, {channels} channels -> 16kHz mono"

3. **Important details from RESEARCH.md:**
   - WASAPI shared mode almost never returns 16kHz — always resample
   - Resampler staging buffer: accumulate raw mono samples, drain chunks of exactly `chunk_size` into `resampler.process()`
   - On flush (recording stops): zero-pad remaining staging samples to chunk_size and process final chunk
   - `try_lock()` not `lock()` in callback to avoid blocking audio thread

4. **write_wav(path: &str, samples: &[f32]) -> Result<(), ...>** function:
   - Use hound::WavWriter with spec: 1 channel, 16000 Hz, 32-bit float
   - Write all samples, finalize

5. **clear_buffer(&self)** method on AudioCapture:
   - Lock buffer, clear it, also reset the resampler staging buffer
