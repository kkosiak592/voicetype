use whisper_rs::{WhisperContext, WhisperContextParameters, FullParams, SamplingStrategy};

// Context creation with GPU

ctx_params.use_gpu(true);

let params = FullParams::new(SamplingStrategy::BeamSearch { beam_size: 5, patience: -1.0 });
params.set_language(Some("en"));

let text = state.full_get_segment_text(0)?;

**Cargo.toml additions:**
```toml
[dependencies]
whisper-rs = { version = "0.15", features = ["cuda"] }
```

Note: The cuda feature compiles CUDA runtime into the binary. This requires:
- CUDA Toolkit 11.7 installed
- CMAKE_CUDA_ARCHITECTURES=61 environment variable set (Pascal/P2000)
- LIBCLANG_PATH set to the LLVM bin directory

If the build fails with clang/bindgen errors, check LIBCLANG_PATH. If it fails with CUDA errors, check nvcc --version and CMAKE_CUDA_ARCHITECTURES. See RESEARCH.md Pitfall 4 and the CUDA Build Environment Setup section.

**Create src-tauri/src/transcribe.rs:**

1. **models_dir() -> PathBuf** function:
   - Read APPDATA env var
   - Return `{APPDATA}/VoiceType/models/`

2. **resolve_model_path() -> Result<PathBuf, String>** function:
   - Build path: `models_dir().join("ggml-large-v3-turbo-q5_0.bin")`
   - Check if file exists with `path.exists()`
   - If not found, return detailed error with:
     a. The path that was checked
     b. Download URL: https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-large-v3-turbo-q5_0.bin
     c. PowerShell download command
   - If found, log the path and file size, return Ok(path)

3. **load_whisper_context(model_path: &str) -> Result<WhisperContext, String>** function:
   - Create WhisperContextParameters::default()
   - Call use_gpu(true)
   - Call WhisperContext::new_with_params(model_path, ctx_params)
   - Log "Whisper model loaded from {path}, GPU enabled"
   - Log model load duration (use Instant::now())

4. **transcribe_audio(ctx: &WhisperContext, audio: &[f32]) -> Result<String, String>** function:
   - Create state via ctx.create_state()
   - Create FullParams with SamplingStrategy::BeamSearch { beam_size: 5, patience: -1.0 }
   - Set language to Some("en") (force English, per user decision)
   - Set temperature to 0.0
   - Set print_special(false), print_progress(false), print_realtime(false), print_timestamps(false)
   - Call state.full(params, audio)
   - Collect all segments into a single string (loop full_n_segments, full_get_segment_text)
   - Trim and return
   - Log transcription duration in milliseconds

5. **Note on WhisperContext thread safety:** WhisperContext must be wrapped in Arc for sharing. Creating a new state per transcription call via ctx.create_state() is safe and the recommended approach (see RESEARCH.md Pitfall 6). Do NOT reuse state across calls.
