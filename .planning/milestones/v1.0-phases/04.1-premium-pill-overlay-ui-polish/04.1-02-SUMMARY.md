## Decisions Made

- `animState` is a ref (not React state) to avoid re-render loops during animation timeout callbacks — displayState drives renders, animState drives CSS class string
- Error dismiss is silent: no `displayState = "error"` assignment — the previous state's content scales down with the pill, matching CONTEXT.md "no visual punishment for failed attempts"
- Success hold is 600ms total (280ms draw + 320ms hold) then triggers exit — gives user time to register success without feeling slow
- Cross-fade uses `pill-content-fade-in` applied to wrapper div on each new content mount — React unmount/mount cycle means previous content disappears as new fades in over 120ms
