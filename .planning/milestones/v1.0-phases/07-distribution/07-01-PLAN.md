   a. `DownloadEvent` enum (serde-serializable, tagged):
      ```rust
      #[derive(Clone, Serialize)]
      #[serde(rename_all = "camelCase", tag = "event", content = "data")]
      pub enum DownloadEvent {
          Started { url: String, total_bytes: u64 },
          Progress { downloaded_bytes: u64, total_bytes: u64 },
          Finished,
          Error { message: String },
      }
      ```

   b. `model_info(model_id)` function returning `(filename, sha256, expected_size_bytes)`:
      - `"large-v3-turbo"` -> `("ggml-large-v3-turbo-q5_0.bin", "394221709cd5ad1f40c46e6031ca61bce88931e6e088c188294c6d5a55ffa7e2", 601_882_624)`
      - `"small-en"` -> `("ggml-small.en-q5_1.bin", "bfdff4894dcb76bbf647d56263ea2a96645423f1669176f4844a1bf8e478ad30", 199_229_440)`
      - Unknown -> None

   c. `download_url(filename)` function returning the HuggingFace resolve URL:
      `format!("https://huggingface.co/ggerganov/whisper.cpp/resolve/main/{}", filename)`

   d. `download_model` Tauri command:
      ```rust
      #[tauri::command]
      pub async fn download_model(
          model_id: String,
          on_event: Channel<DownloadEvent>,
      ) -> Result<(), String>
      ```
      Implementation:
      - Look up model info via `model_info(&model_id)`, return Err if unknown
      - Construct URL via `download_url`
      - Ensure `models_dir()` exists (`tokio::fs::create_dir_all`)
      - Send `Started` event with URL and expected_size_bytes as total_bytes
      - HTTP GET via `reqwest::Client::new().get(url).send().await`
      - Use `response.content_length()` if available, otherwise fall back to `expected_size_bytes` for progress denominator (pitfall 6 from RESEARCH.md)
      - Stream via `response.bytes_stream()` with `futures_util::StreamExt`
      - Write chunks to `dest.with_extension("tmp")` via `tokio::fs::File` + `tokio::io::AsyncWriteExt`
      - Feed each chunk to `sha2::Sha256::new()` hasher via `hasher.update(&chunk)`
      - Send `Progress` events every chunk (throttle not needed — Channel handles backpressure)
      - After stream completes: finalize SHA256, compare hex digest to expected
      - On checksum mismatch: delete .tmp file, return `Err("Checksum mismatch: expected {expected}, got {actual}")`
      - On success: `tokio::fs::rename(tmp_path, dest).await` (atomic move)
      - Send `Finished` event
      - On any error during download: attempt to delete .tmp file, send `Error` event, return Err

   e. Error handling: wrap reqwest errors and IO errors into String via `.map_err(|e| e.to_string())`. Send `Error` event before returning Err so frontend always gets notification.

   IMPORTANT: The `download_model` command must NOT be gated behind `#[cfg(feature = "whisper")]` — it only does HTTP download and file I/O, no whisper-rs dependency. This allows building the download system without CUDA/LLVM. However, it does import `transcribe::models_dir()` which IS gated. Solution: extract `models_dir()` to a standalone function in download.rs (duplicating the simple APPDATA path logic) OR conditionally import. Best approach: duplicate the 3-line `models_dir()` function in download.rs since it's trivial and avoids feature-gate coupling.

   ```rust
   struct FirstRunStatus {
       needs_setup: bool,
       gpu_detected: bool,
       recommended_model: String,
   }

   #[cfg(feature = "whisper")]
   #[tauri::command]
   fn check_first_run() -> FirstRunStatus {
       use crate::transcribe::{detect_gpu, models_dir, ModelMode};
       let gpu_mode = matches!(detect_gpu(), ModelMode::Gpu);
       let dir = models_dir();
       let large_exists = dir.join("ggml-large-v3-turbo-q5_0.bin").exists();
       let small_exists = dir.join("ggml-small.en-q5_1.bin").exists();
       FirstRunStatus {
           needs_setup: !large_exists && !small_exists,
           gpu_detected: gpu_mode,
           recommended_model: if gpu_mode { "large-v3-turbo".to_string() } else { "small-en".to_string() },
       }
   }
   ```
