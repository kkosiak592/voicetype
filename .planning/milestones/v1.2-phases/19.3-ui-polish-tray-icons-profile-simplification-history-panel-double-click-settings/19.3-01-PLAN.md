     Check button == MouseButton::Left to avoid right-double-click opening settings (Pitfall 6 from RESEARCH.md).

  3. Add a new public function `destroy_tray()` to tray.rs:
     ```rust
     /// Destroy the tray icon explicitly before relaunch.
     /// Prevents Windows from showing a stale icon alongside the new process's icon.
     pub fn destroy_tray(app: &tauri::AppHandle) {
         if let Some(tray) = app.tray_by_id("tray") {
             let _ = tray.set_visible(false);
         }
     }
     ```
     Use `set_visible(false)` — the Tauri `TrayIcon` type does not expose a public `destroy()` method. Setting visibility to false removes the icon from the tray. If `TrayIconExt` has a `destroy()` method available in the project's Tauri version, prefer that. Check the available methods on the tray icon type before committing to `set_visible(false)`.

     }
