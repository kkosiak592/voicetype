# Phase 19.2 Plan 01: Remove Distil Large v3.5, Add Benchmark Stats, Recommend Parakeet Summary

**One-liner:** Removed distil-large-v3.5 from all model surfaces, embedded WER/latency/size benchmark stats in descriptions, and made parakeet-tdt-v2-fp32 the universal recommended model.

## What Was Built

Cleaned the model registry from 5 models to 4 by removing the worst-performing option (distil-large-v3.5: 15.6s avg latency, 5.9% WER, 513 MB). All remaining model descriptions now include benchmark-derived stats. Parakeet TDT is the single unconditional recommendation for all users.

## Changes Made

### src-tauri/src/download.rs
- Removed `"distil-large-v3.5"` match arm from `model_info()`
- Updated doc comment to remove distil example reference

### src-tauri/src/lib.rs
- Removed `"distil-large-v3.5"` match arm from `model_id_to_path()` — existing users with distil saved in settings.json get graceful auto-fallback via startup error chain
- Removed distil `ModelInfo` push from `list_models()`
- Updated all 4 model descriptions with benchmark stats (WER, avg latency, size)
- Changed parakeet `recommended` from `gpu_mode` to `true` unconditionally
- Changed small-en `recommended` from `!gpu_mode` to `false`
- Removed unused `gpu_mode` variable and `CachedGpuMode` state access from `list_models()`
- Removed `distil_v35_exists` variable and its `&& !distil_v35_exists` clause from `check_first_run()` needs_setup condition
- Changed `recommended_model` in `check_first_run()` from GPU-conditional to unconditional `"parakeet-tdt-v2-fp32"`

### src/components/FirstRun.tsx
- Removed distil-large-v3.5 entry from MODELS array (5 → 4 entries)
- Reordered: parakeet first (recommended model at top), then moonshine, large-v3-turbo, small-en
- Updated quality strings: parakeet "Best accuracy · works everywhere", moonshine "Fastest · best for short clips", large-v3-turbo "High accuracy · NVIDIA GPU"
