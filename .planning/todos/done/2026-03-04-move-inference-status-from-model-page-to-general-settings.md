## Problem

The inference status indicator currently lives on the Model settings page, but it feels out of place there. The Model page should focus on model selection and configuration, not runtime status.

## Solution

Add a new "System" tab to the settings UI and relocate the inference status display from the Model page into it. This tab can also serve as home for other system-level info in the future.
