## Problem

The inference status indicator currently lives on the Model settings page, but it feels out of place there. The Model page should focus on model selection and configuration, not runtime status. The inference status would be more logically located in the General settings section.

## Solution

Relocate the inference status display from the Model settings page to the General settings page (or another more appropriate top-level location). Ensure it still updates correctly after being moved.
