    - apply("compare values") with {"mpa": "MPa"} returns "compare values" unchanged (no substring match)
    - apply("aci three eighteen is the code") with {"aci three eighteen": "ACI 318"} returns "ACI 318 is the code" (multi-word phrase)
    - apply("why section W8x31") with {"why section": "W-section"} returns "W-section W8x31"
    - CorrectionsEngine::from_map with empty HashMap returns engine that passes text through unchanged
    - structural_engineering_profile() returns Profile with id "structural-engineering", non-empty initial_prompt containing "I-beam, W-section, MPa", non-empty corrections HashMap, all_caps false
    - general_profile() returns Profile with id "general", empty initial_prompt, empty corrections, all_caps false
    - Profile.all_caps=true causes "hello world".to_uppercase() to produce "HELLO WORLD" (tested at integration point, not in profile itself)
