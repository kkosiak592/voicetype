All four phase requirements are fully implemented and wired end-to-end:

**TRAY-FIX / TRAY-DBLCLICK:** tray.rs has both the DoubleClick handler with Left button guard and the destroy_tray helper. lib.rs registers the destroy_tray IPC command. updater.ts calls it in a try/catch before relaunch(). The wiring is complete and correct.

HistoryState is loaded from disk in setup() before build_tray. HistorySection.tsx loads on mount, renders entries with text/timestamp/engine, supports click-to-copy, and handles the empty state.
