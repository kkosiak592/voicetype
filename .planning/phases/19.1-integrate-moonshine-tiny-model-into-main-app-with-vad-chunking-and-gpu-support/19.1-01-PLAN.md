Implement three public functions mirroring `transcribe_parakeet.rs`:

1. `pub fn load_moonshine(model_dir: &std::path::Path, provider: &str) -> Result<transcribe_rs::engines::moonshine::MoonshineEngine, String>`:
   - Create `MoonshineEngine::new()`
   - Create `MoonshineModelParams::tiny()`
   - Set `params.execution_providers` based on `provider`:
     - `"cuda"` -> `Some(vec![ort::execution_providers::CUDAExecutionProvider::default().build(), ort::execution_providers::CPUExecutionProvider::default().build()])`
     - `"directml"` -> `None` (CPU fallback for now — DirectML EP for Moonshine is an open question per research)
     - anything else -> `None` (CPU default)
   - Call `engine.load_model_with_params(model_dir, params)` — map error to String
   - Log load duration at INFO level
   - Return the engine

   IMPORTANT: For the `ort` execution provider types, access them through the `ort` crate which is already a transitive dependency via `transcribe-rs`. However, the main app does not have `dep:ort` enabled in the moonshine feature. Use `transcribe_rs::ort::execution_providers::*` if transcribe-rs re-exports ort, OR add `dep:ort` to the moonshine feature if needed for compilation. 2. `pub fn warm_up_moonshine(engine: &mut transcribe_rs::engines::moonshine::MoonshineEngine)`:
   - Create 8000 zero-samples (0.5s at 16kHz)
   - Call `engine.transcribe_samples(silent_audio, None)`
   - Log warm-up duration, swallow errors (non-fatal)

3. `pub fn transcribe_with_moonshine(engine: &mut transcribe_rs::engines::moonshine::MoonshineEngine, audio: &[f32]) -> Result<String, String>`:
   - Check `audio.len() > 30 * 16000` (30s at 16kHz)
   - If over 30s: call `crate::vad::vad_chunk_for_moonshine(audio)` to split, then loop over chunks calling `engine.transcribe_samples(chunk, None)`, concatenate results with space separator
   - If under 30s: call `engine.transcribe_samples(audio.to_vec(), None)` directly
   - Map `Box<dyn Error>` to `String` via `format!`
   - Log transcription duration and first 80 chars of result
   - Return trimmed text
