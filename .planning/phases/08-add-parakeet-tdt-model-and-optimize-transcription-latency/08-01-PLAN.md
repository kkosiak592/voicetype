1. Add a `PARAKEET_FILES` constant array defining the 5 ONNX model files from `istupakov/parakeet-tdt-0.6b-v2-onnx` on HuggingFace:
   ```rust
   const PARAKEET_FILES: &[(&str, u64)] = &[
       ("encoder-model.int8.onnx", 652_000_000),
       ("decoder_joint-model.int8.onnx", 9_000_000),
       ("nemo128.onnx", 140_000),

   ];
   ```
   NOTE: SHA256 checksums are not available from HuggingFace for these files. Skip SHA256 validation for Parakeet files — validate by expected file size instead (total bytes downloaded must match content-length). Add a TODO comment noting checksums should be added once computed from verified downloads.

2. Add a `parakeet_download_url(filename: &str) -> String` helper:
   ```rust
   fn parakeet_download_url(filename: &str) -> String {
       format!(
           "https://huggingface.co/istupakov/parakeet-tdt-0.6b-v2-onnx/resolve/main/int8/{}",
           filename
       )
   }
   ```

3. Create `download_parakeet_model` Tauri command:
   ```rust
   #[tauri::command]
   pub async fn download_parakeet_model(on_event: Channel<DownloadEvent>) -> Result<(), String>
   ```
   - Create `models_dir().join("parakeet-tdt-v2")` directory
   - Calculate total_bytes as sum of all PARAKEET_FILES expected sizes
   - Send `DownloadEvent::Started { url: "parakeet-tdt-v2 (5 files)", total_bytes }`
   - Loop through PARAKEET_FILES:
     - Download each file using the same streaming pattern as `download_model` (reqwest + bytes_stream)
     - Write to `models_dir()/parakeet-tdt-v2/{filename}` using .tmp-then-rename atomic write
     - Track cumulative downloaded_bytes across all files
     - Send `DownloadEvent::Progress { downloaded_bytes: cumulative, total_bytes }` during each chunk
     - On any file error, delete the entire `parakeet-tdt-v2/` directory and send Error event
   - After all files downloaded, send `DownloadEvent::Finished`

4. Add a `parakeet_model_exists() -> bool` public function:
   - Check if `models_dir().join("parakeet-tdt-v2").join("encoder-model.int8.onnx").exists()`
   - This is sufficient — if the encoder exists, the download completed (atomic write pattern)

5. Keep the existing `download_model` command unchanged — it handles Whisper models only.
